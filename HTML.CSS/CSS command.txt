CSSのコマンド（プロパティやセレクタなど）を分野別に整理して、最低100個を表にまとめたで！CSSはたくさんのプロパティがあって、それぞれが特定の目的で使われる。これを理解して使いこなせば、より細かくデザインを調整できるようになるで。

### 1. **レイアウト関連**
| プロパティ             | 説明 |
|------------------------|------|
| `display`              | 要素の表示方法を指定する。ブロック、インライン、フレックスなど。 |
| `position`             | 要素の位置を指定する（`static`、`relative`、`absolute`、`fixed`、`sticky`）。 |
| `top`                  | 要素の上からの位置を指定する。`position`と一緒に使う。 |
| `right`                | 要素の右からの位置を指定する。`position`と一緒に使う。 |
| `bottom`               | 要素の下からの位置を指定する。`position`と一緒に使う。 |
| `left`                 | 要素の左からの位置を指定する。`position`と一緒に使う。 |
| `float`                | 要素を左または右に回り込ませる。 |
| `clear`                | `float`によって回り込んだ要素を解除する。 |
| `z-index`              | 要素のスタッキング順序（重なり順）を指定する。 |
| `flex-direction`       | フレックスボックスの子要素の配置方向を指定する（行か列か）。 |
| `justify-content`      | フレックスコンテナ内の要素の水平方向の配置を指定する。 |
| `align-items`          | フレックスコンテナ内の要素の垂直方向の配置を指定する。 |
| `align-self`           | 特定のフレックスアイテムの配置を指定する。 |
| `order`                | フレックスアイテムの表示順序を指定する。 |
| `grid-template-columns`| グリッドレイアウトの列のサイズを指定する。 |
| `grid-template-rows`   | グリッドレイアウトの行のサイズを指定する。 |
| `grid-column-gap`      | グリッドの列間のギャップを指定する。 |
| `grid-row-gap`         | グリッドの行間のギャップを指定する。 |
| `grid-area`            | グリッドアイテムの位置とサイズを指定する。 |
| `gap`                  | グリッドやフレックスアイテムの間隔を指定する。 |
| `overflow`             | 要素の内容がはみ出した場合にどう表示するかを指定する（スクロール、非表示など）。 |
| `box-sizing`           | 要素のサイズを決定する際の基準（コンテンツ、パディング、ボーダー）を指定する。 |

### 2. **ボックスモデル関連**
| プロパティ             | 説明 |
|------------------------|------|
| `margin`               | 要素の外側の余白を指定する。 |
| `margin-top`           | 上部の余白を指定する。 |
| `margin-right`         | 右側の余白を指定する。 |
| `margin-bottom`        | 下部の余白を指定する。 |
| `margin-left`          | 左側の余白を指定する。 |
| `padding`              | 要素の内側の余白を指定する。 |
| `padding-top`          | 上部の内側の余白を指定する。 |
| `padding-right`        | 右側の内側の余白を指定する。 |
| `padding-bottom`       | 下部の内側の余白を指定する。 |
| `padding-left`         | 左側の内側の余白を指定する。 |
| `border`               | 要素の境界線を指定する。 |
| `border-width`         | 境界線の幅を指定する。 |
| `border-style`         | 境界線のスタイル（実線、点線など）を指定する。 |
| `border-color`         | 境界線の色を指定する。 |
| `border-radius`        | 要素の角を丸くする。 |
| `width`                | 要素の幅を指定する。 |
| `height`               | 要素の高さを指定する。 |
| `max-width`            | 要素の最大幅を指定する。 |
| `max-height`           | 要素の最大高さを指定する。 |
| `min-width`            | 要素の最小幅を指定する。 |
| `min-height`           | 要素の最小高さを指定する。 |

### 3. **タイポグラフィ関連**
| プロパティ             | 説明 |
|------------------------|------|
| `font-family`          | 使用するフォントを指定する。 |
| `font-size`            | フォントのサイズを指定する。 |
| `font-weight`          | フォントの太さを指定する。 |
| `font-style`           | フォントのスタイル（通常、斜体など）を指定する。 |
| `text-align`           | テキストの整列方法を指定する（左寄せ、中央寄せなど）。 |
| `text-decoration`      | テキストの装飾（下線、打ち消し線など）を指定する。 |
| `text-transform`       | テキストの大文字・小文字変換を指定する。 |
| `line-height`          | 行の高さを指定する。 |
| `letter-spacing`       | 文字間のスペースを指定する。 |
| `word-spacing`         | 単語間のスペースを指定する。 |
| `text-shadow`          | テキストに影をつける。 |
| `white-space`          | 改行やスペースの扱いを指定する。 |
| `direction`            | テキストの書字方向を指定する（左→右、右→左）。 |
| `unicode-bidi`         | 複数言語を混在させる際のテキスト方向の設定。 |
| `letter-spacing`       | 文字の間隔を指定する。 |
| `word-break`           | 単語の途中での改行を指定する。 |

### 4. **カラー関連**
| プロパティ             | 説明 |
|------------------------|------|
| `color`                | テキストの色を指定する。 |
| `background-color`     | 背景色を指定する。 |
| `background-image`     | 背景画像を指定する。 |
| `background-size`      | 背景画像のサイズを指定する。 |
| `background-position`  | 背景画像の位置を指定する。 |
| `background-repeat`    | 背景画像の繰り返し方法を指定する。 |
| `background-attachment`| 背景画像の固定やスクロールを指定する。 |
| `opacity`              | 要素の透明度を指定する。 |
| `box-shadow`           | 要素に影をつける。 |
| `filter`               | 要素に視覚効果を適用する（ぼかし、明るさなど）。 |
| `mix-blend-mode`       | 要素の描画モードを指定する（合成効果）。 |

### 5. **アニメーション関連**
| プロパティ             | 説明 |
|------------------------|------|
| `transition`           | 要素の状態変化にアニメーションをつける。 |
| `transition-duration`  | アニメーションの時間を指定する。 |
| `transition-property`  | どのプロパティに対してアニメーションを適用するか指定する。 |
| `transition-timing-function` | アニメーションの速度曲線を指定する。 |
| `animation`            | カスタムアニメーションを指定する。 |
| `animation-name`       | アニメーションの名前を指定する。 |
| `animation-duration`   | アニメーションの再生時間を指定する。 |
| `animation-timing-function` | アニメーションの速度曲線を指定する。 |
| `animation-delay`      | アニメーションの遅延時間を指定する。 |
| `animation-iteration-count` | アニメーションを何回繰り返すかを指定する。 |
| `animation-direction`  | アニメーションの再生方向を指定する。 |
| `animation-fill-mode`  | アニメーション終了後の状態を指定する。 |



### 6. **トランスフォーム関連**
| プロパティ               | 説明 |
|--------------------------|------|
| `transform`              | 要素に対して2Dまたは3D変換（回転、拡大、移動、傾斜）を適用する。 |
| `transform-origin`       | 変換の基準点を指定する。 |
| `rotate`                 | 要素を回転させる。 |
| `scale`                  | 要素のサイズを拡大・縮小する。 |
| `scaleX`                 | 要素の横方向の拡大・縮小を指定する。 |
| `scaleY`                 | 要素の縦方向の拡大・縮小を指定する。 |
| `translate`              | 要素を移動させる。 |
| `translateX`             | 要素を水平方向に移動させる。 |
| `translateY`             | 要素を垂直方向に移動させる。 |
| `skew`                   | 要素を斜めに傾ける。 |
| `skewX`                  | 要素を水平方向に傾ける。 |
| `skewY`                  | 要素を垂直方向に傾ける。 |
| `perspective`            | 3D変換の視点を指定する。 |
| `perspective-origin`     | 3D変換の視点の基準点を指定する。 |
| `backface-visibility`    | 要素の裏側が表示されるかどうかを指定する。 |
| `rotateX`                | 要素をX軸に沿って3D回転させる。 |
| `rotateY`                | 要素をY軸に沿って3D回転させる。 |
| `rotateZ`                | 要素をZ軸に沿って3D回転させる。 |

### 7. **リスト・テーブル関連**
| プロパティ               | 説明 |
|--------------------------|------|
| `list-style`             | リストのマーカー（番号や記号）を指定する。 |
| `list-style-type`        | リストのマーカーの種類（ディスク、サークル、スクエア、番号など）を指定する。 |
| `list-style-position`    | マーカーの位置（`inside`または`outside`）を指定する。 |
| `list-style-image`       | リストのマーカーとして画像を指定する。 |
| `table-layout`           | テーブルのレイアウト方式を指定する（`fixed`や`auto`）。 |
| `border-collapse`        | テーブルのセルのボーダーを結合するかどうかを指定する。 |
| `border-spacing`         | テーブルセル間のスペースを指定する。 |
| `caption-side`           | テーブルのキャプションの位置を指定する。 |
| `empty-cells`            | 空のセルを表示するか非表示にするかを指定する。 |
| `vertical-align`         | テーブルセルのコンテンツの縦方向の位置を指定する。 |
| `text-align`             | テーブルセルのテキストの水平方向の整列を指定する。 |
| `white-space`            | テキストの折り返しと空白文字の処理方法を指定する。 |
| `border-collapse`        | セルの境界線を結合して表示するか、分離して表示するかを指定する。 |
| `table-layout`           | テーブルのレイアウトを自動で調整するか、固定にするかを指定する。 |


### 8. **背景・ボーダー関連**
| プロパティ               | 説明 |
|--------------------------|------|
| `background`             | 背景に関する複数のプロパティを一括で指定できる。 |
| `background-clip`        | 背景が描画される範囲を指定する（ボーダー、パディングなど）。 |
| `background-origin`      | 背景画像の位置を決定するための基準を指定する。 |
| `background-blend-mode`  | 背景画像と背景色の合成方法を指定する。 |
| `border-image`           | 画像を境界線として使用するためのプロパティ。 |
| `border-image-source`    | ボーダーの画像ソースを指定する。 |
| `border-image-slice`     | ボーダー画像を分割する領域を指定する。 |
| `border-image-width`     | ボーダー画像の幅を指定する。 |
| `border-image-outset`    | ボーダー画像の範囲を指定する。 |
| `border-image-repeat`    | ボーダー画像を繰り返すか、引き伸ばすかを指定する。 |
| `border-top-left-radius` | 左上の角を丸める半径を指定する。 |
| `border-top-right-radius`| 右上の角を丸める半径を指定する。 |
| `border-bottom-left-radius` | 左下の角を丸める半径を指定する。 |
| `border-bottom-right-radius` | 右下の角を丸める半径を指定する。 |

### 9. **セレクタ関連**
| セレクタ                 | 説明 |
|--------------------------|------|
| `*`                      | すべての要素を対象とする。 |
| `element`                | 指定したタグ（要素）を対象とする。 |
| `.class`                 | 特定のクラスを持つ要素を対象とする。 |
| `#id`                    | 特定のIDを持つ要素を対象とする。 |
| `element1, element2`     | 複数の要素を同時に対象とする。 |
| `element element`        | 子孫セレクタ。指定した要素の全ての子孫要素を対象とする。 |
| `element > element`      | 子セレクタ。指定した要素の直接の子要素のみを対象とする。 |
| `element + element`      | 隣接兄弟セレクタ。指定した要素の直後の兄弟要素を対象とする。 |
| `element ~ element`      | 一般兄弟セレクタ。指定した要素の後の兄弟要素すべてを対象とする。 |
| `:first-child`           | 親要素の最初の子要素を対象とする。 |
| `:last-child`            | 親要素の最後の子要素を対象とする。 |
| `:nth-child(n)`          | 親要素のn番目の子要素を対象とする。 |
| `:hover`                 | マウスオーバー時にスタイルを適用する。 |
| `:focus`                 | 要素がフォーカスされたときにスタイルを適用する。 |
| `:active`                | 要素がアクティブ状態（クリック中）にスタイルを適用する。 |
| `:before`                | 要素の内容の前に擬似要素を挿入する。 |
| `:after`                 | 要素の内容の後に擬似要素を挿入する。 |
| `:not(selector)`         | 指定したセレクタに一致しない要素を対象とする。 |
| `[attribute]`            | 指定した属性を持つ要素を対象とする。 |
| `[attribute=value]`      | 指定した属性とその値が一致する要素を対象とする。 |
| `[attribute^=value]`     | 属性値が指定した値で始まる要素を対象とする。 |
| `[attribute$=value]`     | 属性値が指定した値で終わる要素を対象とする。 |
| `[attribute*=value]`     | 属性値が指定した値を含む要素を対象とする。 |

### 10. **レスポンシブデザイン関連**
| プロパティ               | 説明 |
|--------------------------|------|
| `@media`                 | 画面サイズやデバイスの条件に応じて異なるスタイルを適用するメディアクエリを作成する。 |
| `min-width`              | メディアクエリの条件として、画面の最小幅を指定する。 |
| `max-width`              | メディアクエリの条件として、画面の最大幅を指定する。 |
| `min-height`             | メディアクエリの条件として、画面の最小高さを指定する。 |
| `max-height`             | メディアクエリの条件として、画面の最大高さを指定する。 |
| `orientation`            | デバイスの向き（縦向き・横向き）に応じてスタイルを変更する。 |
| `aspect-ratio`           | 画面の縦横比に応じてスタイルを変更する。 |
| `resolution`             | デバイスの解像度に応じてスタイルを変更する。 |

### 11. **リッチメディア関連**
| プロパティ               | 説明 |
|--------------------------|------|
| `object-fit`             | 画像や動画のコンテナ内での表示方法を指定する（`fill`、`contain`、`cover`など）。 |
| `object-position`        | 画像や動画の表示位置を指定する。 |
| `image-rendering`        | 画像のレンダリング方法を指定する。 |
| `video`                  | HTMLの`<video>`要素のスタイルを指定する。 |
| `audio`                  | HTMLの`<audio>`要素のスタイルを指定する。 |

### 12. **その他の便利なプロパティ**
| プロパティ               | 説明 |
|--------------------------|------|
| `cursor`                 | マウスカーソルの形状を指定する。 |
| `visibility`             | 要素を非表示にする（`hidden`）が、レイアウト上には残す。 |
| `content`                | `::before`や`::after`で挿入する擬似要素の内容を指定する。 |
| `clip-path`              | 要素の表示領域をクリップ（切り取り）する。 |
| `user-select`            | テキストの選択を可能または禁止する。 |
| `pointer-events`         | 要素へのクリックやホバーのイベントを無効化する。 |
| `resize`                 | 要素のサイズ変更を可能にする。 |
| `quotes`                 | `q`要素の引用符の形式を指定する。 |
| `will-change`            | 今後変更が予測されるプロパティを事前にブラウザに知らせ、パフォーマンスを向上させる。 |
| `scroll-behavior`        | スクロールの挙動（スムーススクロールなど）を指定する。 |
| `scroll-snap-align`      | 要素をスクロールスナップ領域に揃える。 |
| `writing-mode`           | テキストの書き方向を指定する（縦書き、横書きなど）。 |
